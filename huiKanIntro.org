GitHub 是一个集分享与协作的平台。这么说， 是不是很熟悉？没错，我知道你想说：论坛。 与论坛不同，GitHub 就像二环东路上的怪坡。

一遍一遍地爬怪坡或许是为了强健自己的躯体，或许是享受和挚友一起拼搏的时光，亦或是单单在乎和那一群人在路上的感觉。 车协不单单是一个骑自行车的社团，在这里你可以做得更多。 GitHub 就像怪坡一样，只不过这个怪坡，可以是任何你感兴趣的东西。散落在论坛里各种各样的骑行经验，或许把他们集中起来用更好看的格式排版出来；或许你自己就是公路车迷，年计划，月计划，不如分享出来或许你会找到同道之人；或许你觉得体训的方案可以是这个样子...

或许 GitHub 能把车协人思想碰撞的火花沉淀下来。
