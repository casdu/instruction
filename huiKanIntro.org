GitHub 是程序员们创造软件的地方，但近些年来的发展让 GitHub 不断成长，使用者也在不断拓宽其本身的意义。 GitHub 更像是一个集分享与协作的平台。这么说， 是不是很熟悉？没错，我知道你想说：论坛。

虽然论坛上现在已经有很多“整理帖”，甚至很多版主新上任时也都会整理一遍，但是这些帖子都是很难被“不断更新”的。而在作者毕业后，再想在上面“稍微补充”一点内容，就会变得异常困难，而完全重写一遍的话，会做很多重复工作。而 GitHub ，天生就是可以解决这个问题的。每一个简单的问题，都可以通过很多人的编辑，不断迭代，而臻于完美。

再打个比方，GitHub 就像二环东路上的怪坡。一遍一遍地爬怪坡或许是为了强健自己的躯体，或许是享受和挚友一起拼搏的时光，亦或是单单在乎和那一群人在路上的感觉。 车协不单单是一个骑自行车的社团，在这里你可以做得更多。 GitHub 就像怪坡一样，只不过这个怪坡，可以是任何你感兴趣的东西。散落在论坛里各种各样的骑行经验，或许把他们集中起来用更好看的格式排版出来；或许你自己就是公路车迷，年计划，月计划，不如分享出来或许你会找到同道之人；或许你觉得体训的方案可以是这个样子...
